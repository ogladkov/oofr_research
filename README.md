This script takes data from different sources to make a good-looking research about core financial data of the World and Russia:

1. Macroeconomics
2. Commodities
3. Money market and FOREX
4. Stocks (World)
5. Bonds
6. Stocks (Russia)
7. Loans
8. Derivatives

Prerequisites:
* Flask 1.0.2
* BeautifulSoup 4.7.1
* lxml 4.2.5
* matplotlib 3.0.0
* numpy 1.15.2
* pandas 0.23.4
* selenium 3.14.1
* plotly 3.2.1

Run:
Just run Flask server with start.py
